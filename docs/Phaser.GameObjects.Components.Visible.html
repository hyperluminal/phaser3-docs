Mixin: Visible

Phaser.GameObjects.Components.Visible

Provides methods used for setting the visibility of a Game Object. Should be applied as a mixin and not used directly.
Since:

The visible state of the Game Object. An invisible Game Object will skip rendering, but still process update logic.
