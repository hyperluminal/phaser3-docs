Constructor

new Class(definition)

Creates a new class with the given descriptor. The constructor, defined by the name `initialize`, is an optional function. If unspecified, an anonymous function will be used which calls the parent class (if one exists). You can also use `Extends` and `Mixins` to provide subclassing and inheritance.
Parameters:
Name Type Description
definition Object a dictionary of functions for the class
Example
var MyClass = new Phaser.Class({

         initialize: function() {
             this.foo = 2.0;
         },

         bar: function() {
             return this.foo + 5;
         }
     });
